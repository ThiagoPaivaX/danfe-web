DANFE Processor

Sistema para identificar automaticamente a loja em arquivos DANFE.

Enviar PDF

Selecione um arquivo PDF com uma ou várias NF-e. O sistema irá analisar cada página e inserir o número da loja.

Como funciona?

O sistema lê o endereço da NF-e, compara com a lista de lojas cadastradas e escreve o número correto no PDF.
